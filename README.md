# Example of bare bones SLF4J/Logback

### How to run
`java -jar build\libs\LoggingDemo-0.0.1-SNAPSHOT.jar`

Log file will be at c:/temp/SystemOut.log

### How to rebuild jar
* For windoze:
`gradlew.bat jar`

* For Linux:
`./gradlew jar`

##### Note 1:  You must configure proxy/password if behind corporate firewall.
